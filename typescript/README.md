# TypeScript Exercises

TypeScript implementations of programming exercises and algorithms.

## Setup

```bash
npm install
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
```

## Run

```bash
npm start
```

## Structure

- `src/algorithms/` - Algorithm implementations
- `tests/` - Test files

## Exercises

- **boxPacking.ts** - Rectangle packing algorithm
